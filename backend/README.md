# Backend

FastAPI local backend for ImageGPT.

## Endpoints

- `POST /v1/images/import`
- `POST /v1/ai/analyze`
- `POST /v1/recipe/apply`
- `POST /v1/export`
- `GET /v1/presets`
- `GET /v1/presets/{name}`
- `POST /v1/presets/save`
- `POST /v1/recipe/reset`
- `GET /v1/history`

Run:

```powershell
python run_server.py
```
